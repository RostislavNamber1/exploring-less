Переменные

Объявление переменных

@link-color: #000; 
h1{
  font-size: clamp(2rem, 8vw - 2rem, 5rem);
  color: @link-color;
}

Свойства как переменные

Вы можете легко обрабатывать свойства как переменные, используя $propсинтаксис. Иногда это может сделать ваш код немного легче.

.widget {
  color: #efefef;
  background-color: $color;
}

Будет интерпритированно в этот код

.widget {
  color: #efefef;
  background-color: #efefef;
}

Родительские селекторы

Ссылка на родительские селекторы с помощью & &Оператор представляет родительские селекторы с вложенным правило , и наиболее часто используется при применении класс модифицирующего или псевдо-класса к существующему селектору:

a {
  color: blue;
  &:hover {
    color: green;
  }
}

Приведет к этому

a {
  color: blue;
}

a:hover {
  color: green;
}

Оператор «родительских селекторов» имеет множество применений. Практически в любое время, когда вам нужно, чтобы селекторы вложенных правил были объединены другими способами, кроме тех, которые используются по умолчанию. Например, другое типичное использование &- создание повторяющихся имен классов:

.button {
  &-ok {
    background-image: url("ok.png");
  }
  &-cancel {
    background-image: url("cancel.png");
  }

  &-custom {
    background-image: url("custom.png");
  }
}

Будет таким

.button-ok {
  background-image: url("ok.png");
}
.button-cancel {
  background-image: url("cancel.png");
}
.button-custom {
  background-image: url("custom.png");
}

Несколько &

&может появляться более одного раза в селекторе. Это позволяет многократно ссылаться на родительский селектор, не повторяя его имя.

.link {
  & + & {
    color: red;
  }

  & & {
    color: green;
  }

  && {
    color: blue;
  }

  &, &ish {
    color: cyan;
  }
}

Выведет

.link + .link {
  color: red;
}
.link .link {
  color: green;
}
.link.link {
  color: blue;
}
.link, .linkish {
  color: cyan;
}
